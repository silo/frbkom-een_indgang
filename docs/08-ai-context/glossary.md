# Ordliste / Glossary

| Term (DA) | English Hint | Definition |
| --------- | ------------ | ---------- |
| Arrangement | Event | Offentligt eller privat arrangement ansøgt via platformen |
| Ansøgning | Application | Data og dokumenter sendt af bruger for godkendelse |
| Bruger | User | Citizen or business applicant authenticated via MitID |
| Admin | Admin | Caseworker med rolle-adgang og department ansvar |
| Department | Department | Kommunal afdeling (Byliv og drift etc.) |
| Lokation | Location | Sted hvor arrangement afholdes (foruddefineret eller adresse) |
| Artefakt | Artifact | Visuelt element på arrangementsplan (scene, bod, facilitet) |
| Dokument | Document | PDF eller vedhæftning relateret til ansøgningen |
| Status | Status | Indsendelseslivscyklus: draft, submitted |
| Reviewstatus | Review Status | Global: Ubehandlet, Under behandling, Delvist godkendt, Godkendt, Afvist |
| CPR | CPR | Dansk personnummer (lagres hashed/krypteret) |
| CVR | CVR | Virksomheds identifikation |
| Kommercielt | Commercial | Angiver om arrangement har markedsleje |
| Tilbagevendende | Recurring | Event der gentages med interval |
| BR18 Bilag 11 | BR18 Annex 11 | Brandregulativ dokument reference |
| Arrangementsplan | Event Plan | Visuel grid plan eller uploadet PDF |
| Grid Planner | Grid Planner | Vue komponent til placering af artefakter |
| Audit Log | Audit Log | Registrering af handlinger på ansøgning |
| Rolle | Role | Systemrolle (user/admin) |

Forklaringer bruges af AI til konsistent forståelse. Undgå synonymer i prompts.

# New Step Flow

There is 5 steps, in order and fields in each step in correct order:

### 1. Step : Kontaktoplysninger

- CVR/CPR nummer - required
- Fulde navn - required
- Telefon - required
- Email - required
- Kontaktperson - required
- Checkbox - optional - "Arrangementet er kommercielt og der skal opkræves markedsleje."

### 2. Eventoplysninger

#### Tilgængelighed for event (subtitle)

- Dato fra - required
- Dato til - required

#### Vælg sted (subtitle)

- Søg efter sted (fx "Rådhuspladsen") eller adresse - Autocomplete dropdown - required
  - If choosen a location an image of the location appears on the bottom of the input field

#### Vælg type

- List of badges - required - Multiple selection possible
  - Festival
  - Cirkus
  - Motionsløb/cykelløb
  - Koncert
  - Mad- og loppemarked
  - Vejfest
  - Andet
